Scrum 敏捷開發工作坊

國立陽明交通大學新竹光復校區 | 2023年3月24日

實體活動 職涯發展 敏捷開發 Solution Challenge

關於活動

你有被光速踢過嗎？沒有沒關係~ 那就來讓自己擁有用 Scrum 光速敏捷開發專案的能力吧！

富比世雜誌提到，敏捷已經是軟體開發主流。並且從 2015起，美國政府 IT 系統招標規定，廠商須證明自己具有敏捷和持續交付的能力。在過去 10 年，敏捷已大幅提升軟體開發成功率、改善品質和上市速度。因此，如何讓公司和自己具備這樣能力，已經是刻不容緩的事情！

本次活動中我們將介紹以下項目，幫助初學者有基本認識：

  • Agile Concept (敏捷核心概念)
  • Scrum Framework (Scrum 框架詳解)
  • Dice Game using Scrum (實作 Scrum 的骰子遊戲)
